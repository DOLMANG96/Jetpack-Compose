# Jetpack Compose

Jetpack Compose를 이용한 다이어트 앱 

MVVM(Model-View-ViewModel) 아키텍처 패턴을 적용하여
UI 관련 로직을 ViewModel에서 효과적으로 관리하며
유지보수 용이하게 만들었습니다.
